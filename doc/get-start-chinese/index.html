Quick start

  1. 进入到 Hexo 的目录
    cd <your_blog_dir>
  2. 从 Github 上 clone BOOKCASE
    git clone https://github.com/objchris/hexo-theme-bookcase themes/bookcase
  3. 修改根路径下的 _config.yml ，将主题改为 bookcase
    theme: bookcase
  4. 按照文档配置主题

bookcase/_config.yml

There are 3 config items in bookcase/_config.yml：

  • root
  • indicate
  • search

root

the link of the author name at the bottom of sidebar.

indicate

The first tag displayed by readers after entering the site for the first time, the tag must be specified by the corresponding article. Without configuration, readers can see all articles when they first enter the website.

search

Whether to enable the site search function. The current site search function is integrated with hexo-generator-search. Please configure according to hexo-generator-search before getting start.

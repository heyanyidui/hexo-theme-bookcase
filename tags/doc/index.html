使用文档

配置文件

BOOKCASE 有主题配置文件，其中有三项内容：

  • root
  • indicate
  • search

root

指代左侧导航栏下方，作者名称对应的超链接

indicate

读者首次进入网站后展示的第一个标签，该标签必须有对应的文章指定。若不指定，则读者首次进入网站时可以看到所有的文章

search

是否开启站内搜索功能，当前的站内搜索功能集成的是 hexo-generator-search。开启前请先按照 hexo-generator-search 进行配置

建议

  • 标签的置顶文章（将时间调整至100年后）做成这个标签所有文章的一个目录，这样做是为了使用手机的用户着想，因为在手机上操作，点击标签跳转到新页面默认是看不到文章列表的，此时有了目录，读者可以直接通过目录查看当前标签下有什么文章，自然就舒服多了。
